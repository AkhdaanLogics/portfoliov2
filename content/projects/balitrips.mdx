## Overview

A beautiful and modern landing page for Bali travel packages. The site showcases popular Bali destinations with stunning visuals, smooth animations, and an intuitive user interface designed to inspire travelers and facilitate bookings.

<ProjectSlider />

## Key Features

- **Stunning Hero Section**: Eye-catching landing page with hero image and call-to-action
- **Destination Gallery**: Interactive cards displaying popular Bali attractions
- **Package Showcase**: Featured travel packages with pricing and itineraries
- **Booking Interface**: Smooth user flow for package selection and booking
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **Smooth Animations**: GSAP animations for engaging user experience
- **Tourist Reviews**: Customer testimonials and ratings section
- **Contact Section**: Easy way to get in touch for inquiries

## Technology Stack

- **Frontend**: React & Next.js
- **Styling**: Tailwind CSS
- **Animations**: GSAP & Framer Motion
- **UI Components**: Custom React components
- **Deployment**: Vercel

## Design Highlights

The landing page features a modern, clean design with:

- Gradient backgrounds and smooth transitions
- Optimized images of Bali's beautiful locations
- Interactive hover effects on destination cards
- Scroll-triggered animations for visual interest
- Fully accessible navigation and forms
- Mobile-first responsive approach
