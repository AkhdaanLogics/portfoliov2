## Overview

AMIKOM Tools is a comprehensive platform designed specifically to help students manage their academic needs more efficiently. The platform provides various practical tools in one easily accessible place, all completely free and privacy-focused.

<ProjectSlider />

## Key Features

- **GPA Calculator**: Real-time GPA calculation using AMIKOM grading scale (A, B, C, D, E) with instant predictions and grade distribution visualization
- **QR Code Generator**: Generate customizable QR codes for links, contact info, or any text with adjustable size and error correction levels
- **Report Templates**: Ready-to-use academic report templates in both Word (.docx) and Google Docs formats for various assignment types
- **Question Bank**: Collection of exam questions and practice materials from various courses across different faculties (FIK, FES, FST)
- **Lecturer Info**: Comprehensive lecturer information per study program with direct links to official faculty websites
- **PDF Merger**: Merge multiple PDF files with intuitive drag-and-drop interface and custom file ordering

## Technology Stack

- **Next.js**: React framework for server-side rendering and static site generation
- **Firebase**: Backend services for authentication and database
- **Firestore**: NoSQL database for real-time data
- **Firebase Auth**: Secure authentication system
- **Tailwind CSS**: Utility-first CSS framework for styling
- **TypeScript**: Type safety and better developer experience
- **QR Code Library**: For generating QR codes
- **PDF.js / PDF-lib**: For PDF manipulation and merging
- **Vercel**: Deployment and hosting

## Highlights

**Free & Open Access**
The platform is completely free for all students without complex registration requirements.

**Privacy First**
Data is not stored on servers, only in the browser. Calculations are done client-side to maintain user privacy.

**Real-time Calculation**
GPA results are calculated instantly as users input data without reload or submit.

**Responsive Design**
Optimal display across all devices - desktop, tablet, and mobile.

**User-Friendly Interface**
Clean and intuitive design with dark theme that's easy on the eyes.

## Challenges & Learning

Building AMIKOM Tools provided valuable learning experiences:

- Implementing real-time calculation with React state management
- Firebase integration for authentication and data persistence
- Client-side PDF manipulation for privacy
- Responsive design with Tailwind CSS
- Performance optimization for Next.js app
- User experience design for academic tools

The platform continues to be developed with new features based on user feedback to provide maximum value to students.
