## Overview

A comprehensive web application for real-time gold price tracking and AI-powered price predictions. Built with machine learning to help investors and traders make informed decisions.

<ProjectSlider />

## Key Features

- **Real-time Price Tracking**: Live gold prices updated every minute
- **AI-Powered Predictions**: Machine learning models forecast future price trends
- **Historical Data Analysis**: Interactive charts showing price history
- **Smart Alerts**: Get notified when prices hit your target levels
- **Portfolio Management**: Track your gold investments

## Technology Stack

- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Node.js, Python
- **ML Models**: TensorFlow, Scikit-learn
- **Database**: PostgreSQL
- **Deployment**: Vercel
