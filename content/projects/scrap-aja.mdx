## Overview

Scrap Aja is a no-backend scraping app that runs entirely in the browser. Paste a URL and get a concise summary of meta, heading structure, key links, and text previews for quick evaluation.

<ProjectSlider />

## Key Features

- **Single URL Scrape**: Fast analysis for a single page in one click
- **Bulk URL**: Process a list of URLs for quick research
- **Meta Focus**: Capture title, description, and OG data
- **Content Structure**: Primary headings to understand content flow
- **Key Links**: Collect important links without manual scrolling
- **Text Preview**: Text summary for quick evaluation
- **Extra Modes**: Full text mode and top keywords

## Technology Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Scraping**: Playwright (no backend)
- **Deployment**: Vercel
